# Docs

`docs/`는 Query Forge의 설계, API, UI, 실험 문서를 정리하는 공간이다. 구현 범위가 넓어질수록 이 디렉터리가 운영 문서의 기준점이 된다.

## 디렉터리 구성

```text
api/           HTTP API 문서
architecture/  저장 구조와 실행 구조 문서
ui/            관리자 화면 설계 문서
experiments/   실험 설계와 리포트 자리
```

## 권장 읽기 순서

1. `architecture/overview.md`
2. `api/README.md`
3. `ui/admin_backoffice.md`
4. 필요 시 세부 API와 아키텍처 문서
